Odoo Module: Product Category Restriction

Overview Features Installation Screenshots

Overview

This Odoo module restricts users from viewing certain products based on their category or sub-category, with full multi-company functionality. Enhance your product visibility control and streamline user access with this module.

Features

  • Restrict product visibility by category and sub-category.
  • Seamless integration with Odoo multi-company setup.
  • Customizable access rules based on user roles.
  • Easy configuration and management.

Installation

To install this module, follow these steps:

  1. Download the module and place it in your Odoo addons directory.
  2. Navigate to Apps in Odoo and click on 'Update Apps List'.
  3. Search for 'Product Category Restriction' and click 'Install'.
